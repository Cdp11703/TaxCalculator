The application is designed using the procedural programming approach. Additionally,
adjustments were made to the income tax calculations. In the first version of the tax calculator,
the income tax was computed using tax rates from the website Tax Calculator Philippines.
However, with the introduction of the Tax Reform for Acceleration and Inclusion (TRAIN) Law
on January 1, 2023, the researchers adjusted the income tax calculation in the tax calculator to
reflect the new regulations.
